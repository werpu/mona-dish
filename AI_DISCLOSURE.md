# AI-Generated Code Disclosure

This project uses AI-assisted tooling (Claude Code / Anthropic Claude) to generate portions of its
source code and test suite, beginning with the TypeScript 6 migration.

## Scope

The following categories of files may contain AI-generated or AI-assisted content:

- TypeScript source modules under `src/main/typescript/`
- Mocha/Chai test files under `src/test/typescript/`

The `dist/` directory is **excluded** — its contents are entirely auto-generated by the build
pipeline and carry no independent authorship.

Individual AI-generated files carry an inline header comment identifying them as such
(see *File Header Convention* below).

## Review and Accountability

All AI-generated code is reviewed by a human developer before being committed to the repository.
The author of record for any commit remains the committing developer, who takes full responsibility
for the correctness, security, and license-compatibility of the content.

## File Header Convention

AI-generated or AI-assisted TypeScript files carry the following header directly below the
Apache 2.0 license block:

```
/*
 * AI-GENERATED FILE
 * This file was generated with the assistance of Claude Code (Anthropic).
 * Generated on: <ISO-8601 date>
 * Reviewed by:  <author>
 *
 * The content has been reviewed and approved by a human developer.
 * All standard project quality and license requirements apply.
 */
```
